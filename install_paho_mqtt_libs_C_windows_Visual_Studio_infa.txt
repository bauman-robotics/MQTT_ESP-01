Установка paho.mqtt.c

вызываем командную строку от имени администратора, заходим в папку paho.mqtt.c
выполняем команду:
cmake --build build --target install

Это установит paho.mqtt.c в C:\Program Files(x86)\Eclipse Paho C\

----
Установка paho.mqtt.cpp

Процедура по своей природе такая же: откройте графический интерфейс CMake, выберите папку, содержащую исходный код, затем папку сборки и нажмите «Настроить».
Теперь, поскольку paho.mqtt.cpp использует библиотеки из paho.mqtt.c, вы должны указать cmake, где найти соответствующие библиотеки paho.mqtt.c.
Для этого настройте переменные PAHO_MQTT_C_INCLUDE_DIRSи PAHO_MQTT_C_LIBRARIES.
PAHO_MQTT_C_INCLUDE_DIRS должен указывать на папку «include» внутри установки paho.mqtt.c, в моем случае:C:/Program Files (x86)/Eclipse Paho C/include
PAHO_MQTT_C_LIBRARIES Я настроил так, чтобы указать на paho-mqtt3c.dll, в моем случае:C:/Program Files (x86)/Eclipse Paho C/bin/paho-mqtt3c.dll
Остальные варианты я оставил нетронутыми.
Наконец, вернитесь в командную строку разработчика, перейдите в папку paho.mqtt.cpp и запустите cmake --build build --target install.
Если все пойдет хорошо, paho.mqtt.cpp будет установлен в C:/Program Files (x86)/paho-mqtt-cpp, в соответствии с переменной конфигурации CMAKE_INSTALL_PREFIX.

C:\Program Files (x86)\Eclipse Paho C
C:\Program Files (x86)\paho-mqtt-cpp
Это установит paho.mqtt.c в C:\Program Files(x86)\paho-mqtt-cpp
